## Wynncraft Status API
The status API returns useful information about the status of the servers.

## Player Count

### Example Request
`http://api.wynncraft.com/public_api.php?action=onlinePlayersSum`

### Example Response
```json
{
    "players_online": 2228,
    "request": {
        "timestamp": 1401914961,
        "ip": "127.0.0.1"
    }
}
```
